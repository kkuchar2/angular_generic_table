2. Fixed / capped height

maxHeight="260px" caps the scroll body: it grows with content up to the cap, then scrolls under a sticky header. Use height instead for an always-fixed height.

3. Fill available height

heightMode="fill" takes the space left in a sized flex column. This panel is a fixed-height box with a heading; the table fills the remainder and scrolls.

Fixed 320px panel — the table fills what's left.

4. Custom cell templates

Multiple projected templates: an initials avatar, a mailto link, and a status badge.

5. Empty state

A centered message replaces the rows when there is no data.
