Generic Table

Generic table component — variants showcase.

1. Full featured

Sorting, pagination, column toggle chips, a custom status badge cell, and clickable rows.

{{ $any(row).status }} @if (selectedRow(); as selected) {

Selected: {{ selected.name }} ({{ selected.email }})

}

2. Scrollable (no pagination)

All rows in a fixed-height, scrollable body with a sticky header. Column toggle disabled.

3. Custom cell templates

Multiple projected templates: an initials avatar, a mailto link, and a status badge.

{{ initials($any(row)) }} {{ $any(row).name }} {{ $any(row).email }} {{ $any(row).status }}

4. Empty state

A centered message replaces the rows when there is no data.
